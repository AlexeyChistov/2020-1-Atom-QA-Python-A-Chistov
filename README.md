# Итоговый проект
В данном проекте была реализована возможность запустить тесты на Jenkins. В проекте присутствует docker-compose.yaml, который поднимает нам базу данных
собирает mock, поднимает selenoid, и в конечном итоге, при успешном поднятии предыдущих - поднимает приложение. 
Тесты поднимаются в Jenkins через отдельный контейнер.
Обнаруженные баги:
* Некорректные редиректы с кнопок Python и Python History (уводят с главной страницы)
* Centos != Fedora
* При успешной регистрации, пользователя переводят на главную страницу, start_active_time не проставляется в 1
